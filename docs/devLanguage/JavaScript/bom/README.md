# bom是什么
`bom`即浏览器对象模型(Browser Object Model)，包括以下几个部分

+ [window(窗口)](#window)
+ [navigator(浏览器)](#navigator)
+ [screen(屏幕)](#screen)
+ [history(访问历史)](#history)
+ [location(浏览器地址)](#location)
